本文记述Linux下的cp/rm/mv强制覆盖以及在Java中的调用（博客园文章迁移）

一、Linux下的cp/rm/mv强制覆盖

1. 反斜杠（\）临时取消别名

2. unalias 取消别名

注意：这只是临时取消cp的别名，不是永久的

3. 修改默认配置文件

输入alias命令，看到系统内部使用的是cp、mv、rm -i 所以怎么输入都是提示覆盖。

修改 ~/.bashrc ，在 “alias cp=’cp -i’ ”前添加#号注释后即可。

重启或者执行 source ~/.bashrc 让命令生效
